This folder contains the code necessary to run the algorithm EmpIRE.

mrp.py is the class implementation of a Markov reward process

mdp.py is the class implementation of a Markov decision process

SA_early_stopping.py is the class implementation of stochastic approximation methods. It contains several algorithms for running SA, as well as procedures specific to MRPs (MRP_SAmethods) and MDPs (MDP_SAmethods).
